Qué es y Qué NO es

jQuery

NO

  • Es un software
  • Es un lenguaje de programación
  • Sustituye de JavaScript

SI

  • Es una librería JavaScript
  • Es libre de usar
  • Mejora nuestra programación JavaScript
  • Creada inicialmente por John Resig
  • Presentada el 14 de enero de 2006 en el BarCamp NYC
  • Es una de las librerías JavaScript más utilizadas
  • Rápida, pequeña y rica en funciones
  • Más simple de usar que cualquier API
  • Facilita el CrossBrowser
  • Sirve para recorrer y manipular el DOM de HTML
  • Manejo de eventos
  • Animación
  • AJAX

Con una combinación de versatilidad y capacidad de ampliación, jQuery ha cambiado la forma en que millones de personas escriben JavaScript
